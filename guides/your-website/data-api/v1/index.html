Data feed API v1

This version is being deprecated. Please update to version 2

The Hystreet data API is a way for 3rd party website developers to access the data within our system in their apps. The most common use of the API is to provide a listing of live properties on a marketing site.

Caveat: The API currently only supports propertiesâ€™ listings. If you need any other data, please get in contact with our support team.

Getting started

Authentication

In order to use our API you will need an authentication token. This can be obtained by speaking to our support team. Tokens need to be passed in as an Authorization request header, with the value as Bearer <authtoken>

For example:

curl -H "Authorization: Bearer abc123" https://demo.hystreet.co.uk/data-feed/v1/properties

Endpoints

The API URLs are relative to https://<your hystreet domain>.hystreet.co.uk/data-feed/v1

For example:

https://demo.hystreet.co.uk/data-feed/v1/properties

Output

The output is in JSON format. We have strived to keep the format as similar as possible to the RTDF format, but that does not necessarily make it the same.

Properties

/properties returns a list of your published properties.

Request example:

curl -H "Authorization: Bearer abc123" https://demo.hystreet.co.uk/data-feed/v1/properties

Most data is populated as the client or the agent completes the property listing process.

Images are found in the media section.
