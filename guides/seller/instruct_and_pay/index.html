Instructing to Sell

From the account page, click “manage my sale”. This will lead you to a page showing all the steps involved in selling a property.

Once an agent has been to the property and completed a valuation, the seller will be prompted to instruct the agent to start selling the property. They just need to click the button to make the instruction.

Sellers will also see your custom sales points here

The following page will allow you to choose which addons to include in the instruction (prices for these can be set through the admin section). Check the box to accept the terms and conditions then hit continue at the bottom of the page. Next, you’ll see the fields to enter card details for paying for the instruction.

This is where payment details are recorded

TIP: If you’re an agent doing a trial run of the consumer site, make sure you are using the test stripe keys and card details. Read our guide on setting up Stripe to find out how to do this.
