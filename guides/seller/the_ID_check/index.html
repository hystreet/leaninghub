The ID Check

This section of the 'Getting To Market Checklist' is a legal requirement- the first page the seller sees after clicking 'complete ID check' is in line with the Money Laundering Regulations, 2007, and should be read carefully. Once they’ve read through everything, they should check the box to show they understand what the ID check entails then click ‘let’s start the ID check’.

The seller then needs to fill out all the information on the following pages. Once everything has been filled out, the seller can initiate the ID check by clicking 'submit for ID check'.

The check - which is powered by Experian - will run behind the scenes on the provided information. The next page will display the Experian response. Once the ID has been accepted, the sale can move forward.

The ID check system provides an instant response to your clients
