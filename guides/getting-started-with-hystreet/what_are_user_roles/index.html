What are User Roles?

You can limit the amount of control your staff members have over the Hystreet platform by setting them as different roles. There are three available but most agencies only require ‘Staff’ and ‘Admin’.

Staff

These users can:

  • Create/Update properties
  • Create/Update sellers
  • Create/Update buyers
  • Book appointments
  • Update Agent availability
Admin

Admin users have all of the rights that Staff users do plus:

  • Set agency wide settings
  • Configure Branches
  • Configure Agents
  • Configure Staff Accounts
