Stripe Information

We've got quite a lot to tell you about Stripe - as such, we've put together a whole guide on the topic of Stripe. you can find our Stripe Guide here.

We use Stripe.com for processing your payments, as one of the largest online payment providers in the world they are both reliable and competitively priced. When you sign up for your Stripe account you’ll be provided with both a set of test keys and a set of live keys. Think of these keys like a deposit box that allows the system to process those payments on your behalf.

You'll see two fields in the branch settings page to enter your secret and publishable keys (usually of the format sk_xxxx_xxxx and pk_xxxx_xxxx, respectively).

Access the admin pages using these buttons.

Test Keys

If you enter your test keys here (usually of the format sk_test_xxxx and pk_test_xxxx) you’ll be able to see how the system handles different responses for you using the test card numbers. For just running through some test customers and doing training, the card number you are looking for is 4242 4242 4242 4242. With that card you can use any 3 digit CVC and any expiry date that is in the future.

Live Keys

If you enter your live keys here (usually of the format sk_live_xxxx and pk_test_xxxx) you’ll be all set to start taking real world payments from customers!
