Using Intercom

Intercom is a 3rd party system that allows you to have a live-chat function on your Hystreet platform. If you're set up with Intercom you'll see a button on your platform that looks something like this:

your clients will click here to start message you instantly

If this is something that would like to include in your online service, you can register for intercom here. As part of your intercom registration, you be given an App ID to enter on the Admin settings page.

You can also find Intercom's guide for finding your App ID here.

Enter your intercom credentials here to use the chat system

If you need more assistance with this feature at any time then get in touch with your Customer Success Manager who will be able to help you out.
