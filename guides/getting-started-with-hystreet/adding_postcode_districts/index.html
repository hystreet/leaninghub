Adding Postcode Districts

You can assign each agent a set of postcode districts. This allows the system to automatically assign any new customers to the right agent for you. To add a new postcode district, start typing the desired postcode into the ‘Add postcode district’ field and a list will appear for you to pick from. Click ‘Save’ and you’ll see the postcode district added to the list.

This is where you'll set your agents' postcodes

Removing a Postcode District

To remove a postcode district just click the cross next to the district you would like to remove. No need to save on this occasion, the district will disappear straight away.

Please note, once you have a assigned a postcode to one agent, you won't then be able to assign it to a second agent. If you'd like to move a postcode to one agent to another, you must first remove it from the original agent before you can add it to the new agent's covered districts.
