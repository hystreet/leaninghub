Adding a New Staff Account

To add an account click the ‘Add account’ button, complete the following fields and then click the ‘Save’ button.

name

This is for your reference so you know which account is which in the main list.

Email

Enter the staff member’s email address, this will act as their username for logging into the system.

Password

There are no restrictions on the password complexity, but we recommend using a strong password. Here is some Guidance on choosing strong passwords and their importance.

Agent

If this user is also an Agent, then select the corresponding Agent in this dropdown list. This enables the system to automatically associate their calendar to their login and to show their appointments on their dashboard. If you can’t see the right Agent here then check that they have been added as a new agent.

Role

Here you can specify which Role this user has. You can find out more about user roles here.
