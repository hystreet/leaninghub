Setting Up Your First Agent

Initially under the ‘Agent’ button in your Admin area you’ll just see a box saying ‘No Agents’, later on this will show a list of any Agents you have configured.

TIP: This section sets up an Agent, if you need them to access the staff site you’ll need to add a Staff Login as well.

your agents list will look like this before you add agents

Either click the ‘Add Agent’ button to add an Agent or to edit an existing Agent simply click the ‘View’ button next to the Agent you want to update. You’ll see the same fields for both.

Agent Details

The first page is where you enter the details for the agent.

Active Checkbox

You can change the active status of an Agent at any time, simply check or uncheck this box and click ‘Save’. Please note that the postcodes associated with that Agent will only be available to book valuations on if the Agent is marked as ‘Active’.

Contact Details

You can enter (or update) the following contact details for the Agent to be used on the website and in emails:

  • Agent Forename
  • Agent Surname
  • Contact Number
  • Email Address
Bio & Experience

Here you should enter a little bit about the agent to help your customers get to know them. This is used on the introduction page as part of booking a valuation. Here you can see the fields side by side:

This is how your details will be displayed for your clients

Branch

Select the Agent’s corresponding branch from the dropdown here. If you can’t see the right branch then check you have correctly entered your branch information.

The Agent’s Photo

To give that truly personal feel to customer’s using your website, you can upload a photo of the Agent on this page. It isn’t mandatory to upload a photo but we strongly recommend that you upload a photo for each of your agents as we’ve found this works really well for both vendors and buyers.

If you can’t see the option to upload a photo then you may not have saved your new Agent yet. Click the ‘Save’ button and the option will appear.
