How to book a valuation

To book a valuation on a property, you will first need to go to properties > add property.

Record the associated branch and agent using the dropdown fields on the left and enter the address using the fields on the right, then hit save.

On the next page, record the valuation details and save again.

Once the property has been added and the valuation details have been entered, click appointments > valuation > add valuation.

Select the agent that is going to carry out the valuation, then select the date and time for the appointment.

When selecting a time for the appointment from the staff site, you will be shown which time slots are available or already booked, but you can book the appointment in any slot.
