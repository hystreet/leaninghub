Managing viewings by both agent and seller

The ‘viewings by: both agent and seller’ works by compiling the availabilities for both the agent and the seller tied to the property.

When a buyer books a viewing on the property from the consumer site, they will only be presented with time slots where both the agent and the seller are available.
