Sending Booking Links to Clients

Once you have put a property on the market, you can expect potnetial buyers to start enquiring. Regardless of whether these enquiries come from the portals, your marketing website or social media, you can easily direct them to viewings using the booking link for that property.

You can find these Links by clicking Properties > View (on the property) > Listing. This will lead you to the listing presentation page.

On the presentation page, you'll see a box labelled 'booking links':

Simply copy and paste the web address in this box

Simply copy and paste the line of text in this box that says [yourbrand].hystreet.co.uk/properties/[ID]/request-viewing. Send the link to the enquirers - following the link will lead them straight to the valuation booking page.

Please note that you only need to send the top line in this box to your clients. If you copy and paste the line underneath the web address as well, the link may not work.

TIP: if you are sending this booking link out to anyone, please make sure that availability has been set for the property. For more information on setting availability, please visit our appointments guide.
