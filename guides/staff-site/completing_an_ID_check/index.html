Completing An ID Check

ID checks can be started from both the staff and consumer site, and are completed using an Experian check service. To start an ID check from the consumer site, click ‘start ID check’ in the top left of the page. On the following page, you’ll see a space where all the IDs being checked will be listed. To add an ID, hit ‘add ID’.

Use these buttons to start a new check and add IDs

This will bring up the necessary fields for the check. You’ll also see a state field- in some cases, you might not need to run the check (i.e. if you have previously completed the check outside of the platform). In these cases, you can set the state field to ‘superceded’. Otherwise, you can leave the field blank for now.

When the form has been filled out, hit save. You’ll be taken back to the list of IDs from the page before, where’ll you’ll see the ID state marked as ‘pending’.

The state of the ID check will be displayed like this

To run the ID check, click view to go back into the ID page, then view again on the ID you are checking and click ‘run ID check’. The ID list will then show the updated state of the check.

Click this to run the check. The state will show the check results
