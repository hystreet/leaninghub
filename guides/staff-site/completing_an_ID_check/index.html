Completing An ID Check

ID checks can be started from both the staff and consumer site, and are completed using an Experian check service. To start an ID check from the staff site, go to Properties & View (on the relevant property) & Compliance & ID . click ‘Add an ID check’ at the top of the page.

Use this button to add a new ID check

On the following page, you’ll see a space where all the IDs being checked will be listed. To add an ID, hit ‘add a person’.

This will take you to a page to enter you seller's information required by Experian for the check. You'll also see a box to enter a 'state'. Use this box to manually set the ID state as 'approved' if you don't need to run an ID check through experian (if you've already completed one in office, for example).

When the form has been filled out, hit save. You’ll be taken back to the list of IDs from the page before, where’ll you’ll see the ID state marked as ‘pending’.

The state of the ID check will be displayed like this

To run the ID check, click view to go back into the ID page, and click ‘run ID check’. The check will run intantly, and the ID list will then show the updated state of the check.

Click this to run the check. The state will show the check results
