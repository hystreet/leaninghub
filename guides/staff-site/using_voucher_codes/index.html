Using Voucher Codes

Many agents like to add voucher codes to their marketing materials to attract new sellers. If this is something you'd like to do, get in touch using the support tab.

We will happily go behind the scenes and set up a voucher code for you. All you need to do is tell us the following bits of information:

  • Your desired code- for example, if you want offer a Â£200 discount on your intructions through the summer, your code could be 'SUMMER200'.
  • Your discount value- how much do you want to reduce your normal price by?
  • Voucher end date- When do you want to run this special offer until?

Once your voucher codes are set up, all you need to do is put them on your marketing materials (be that on flyers, newspaper articles, tv ads, or anywhere else) and start promoting your service.

It's also worth noting we can add as many vouchers as you like. You could have a different voucher code for every marketing campaign you run, and see which returns the most interest.
