Search and Filtering on the Property List

Search and filtering for the property list is currently undergoing user testing, and as such is only available to a small number of agencies. Once this user testing has been complete, search and filtering for the property list will become available on all Hystreet platforms.

You can now find properties in the property list by using the search and filter options.

you'll see these options at the top of your property list

Filtering by branch or negotiatior

You can filter the property list to show propertiies for a specific agent or branch. These are drop-down fields, so clicking on the boxes will bring up a list of all agents or branches on your platform. You can also check multiple items on these drop-downs (to show a select group of agents' properties, for example).

specify branches or agents here

Filtering by statuses

You can filter the property list by sale statuses. The filterable states are 'Potential', 'On Market', 'Under offer', and 'SSTC'. As with filtering by branch or negotiator, you can select multiple statuses to filter by; You could see a list of all properties under offer or SSTC, for instance.

use this drop-down to filter the list by sale states

Searching by Addresses or seller

You can find a specific property quickly by searching for the address or seller. If you're using the address to find the property, start typing in the 'search by address' box. Likewise, if you're using the seller's name, start typing in the 'search by seller' box.

start typing in these boxes to find properties

Clearing filters

To revert all filtering and see your full stock of properties on your platform, simply click the 'clear filters' button.

click this to clear all filters applied to the property list
