Registering a New Property

In the properties section, you’ll see a table with all properties added through both the staff and consumer site. The left column shows the status of the property (Potential, On Market, Under Offer, SSTC or Withdrawn). You'll also see columns for the property address and the seller's name. To the right of each entry on the table you’ll see ‘view’ button. This will take you into the details section for that property.

If this list of properties exceeds ten entries, it will be split up into multiple pages. You’ll see page numbers underneath the list to move between the pages.

Use this bar to view the next page of properties

Above and to the right of the list, there’s an ‘add property’ button. Clicking this takes you to a page to fill out the new property's address and select the branch and agent tied to the property. Fill out these details then click ‘save’ to add the property to the property list.
