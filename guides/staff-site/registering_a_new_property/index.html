Registering a New Property

Click this button to add a new property

To add a new property, go to the 'Properties' page and click the 'Add property' button.

On the following page, use the boxes to select the branch and agent tied to the property. Next, enter the property's postcode and click 'Find Address'. The system will bring up a list of all addresses in that postcode - just click the correct one.

On the odd occasion (if you are adding a new build, for example), the system might not find the property address in the database. Don't worry if this happens, you can still manually enter the postcode by clicking the link that says "No postcode? Add the address".

When the address has been entered, you just need to hit save. This will add the property to your system and take you to the property details page, where you can record information about the property before the valuation. You can read our guide on entering pre-valuation details here.
