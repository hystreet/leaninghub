Adding Rooms to a Listing

You can add new rooms to a listing by going to Properties > View (on the property) > Listing > Rooms.

This tab will take you to the Rooms page

From here, click the ‘add room’ button at the top of the page. On the next page, you'll see four fields:

Provide details for the room here

  • Name: This would typically be something like 'Master Bedroom' or 'Guest En-suite'.
  • Dimensions: Enter the room measurements as per the floorplan here.
  • Room Type: This drop-down menu will determine which logo shows next to the room name and dimensions on the listing from the consumer site.
  • Description: Use this space to write a description of the room in addition to the main property description.

Repeat this process until you've added every room you'd like featured on the listing. Each room you've added will then be shown on the consumer site, on the left side of the listing just below the property description.
