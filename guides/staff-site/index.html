Main Navigation

On signing in, you’ll be taken straight to the dashboard page. Use the bars at the top of the page to navigate around the site.

The two buttons on the top bar say ‘admin’ and ‘logout’. They take you to the admin section and log you out of the staff site, respectively. The admin section is covered here. On the second row, you’ll see the links to the ‘dashboard’, ‘properties’, ‘buyers’ and ‘diary’ sections.

When viewing a property (we’ll get to that shortly) you’ll see another bar appear, with links to ‘details’, ‘listing’, ‘compliance’, ‘appointments’ and ‘offers’.

The Properties List

In the properties section, you’ll see a table with all properties added through both the staff and consumer site. The left column shows the seller’s name and the centre shows the property address. To the right of each entry on the table you’ll see ‘view’ button, which will take you into the details section for that property.

If this list of properties exceeds ten entries, it will be split up into multiple pages. you’ll see page numbers underneath the list to move between the pages.

Above and to the right of the list, there’s an ‘add property’ button. Clicking this takes you to a page to fill out the new properties address, and use the two drop-down fields to select the branch and agent tied to the property. Filling out these details then clicking ‘save’ will add the property to the property list.

Details Section

above the main body of the page, you’ll see a set of tabs. Use these to move between the subsections of the details section.

Valuation Details

When clicking view on a property in the property list, you’ll immediately be brought here. This page shows all the information from when the property was added, but you’ll also see the fields for recording any information gathered at the valuation. These fields look like this:

Seller Details

You can tie sellers to the added property here. Multiple sellers can be added by clicking the ‘add seller’ button on the top right of the page. Doing so will bring up an empty seller details field, where you can add the name, email address and contact number for the seller.

To make sure that sellers aren’t accidently duplicated, the system checks to make sure the emails are valid and have not already been recorded elsewhere. If clicking save doesn’t successfully add the seller, please check that the email has correctly been entered.

Valuation Reports

Once an agent has completed a valuation on the property. The highest, lowest, and recommended values are recorded here. Once the fields have been filled and saved, hit send report. This will send the values over to the consumer site, so the buyer can then make a decision on instructing the agent to sell their home.

Taking payment

The payments page shows a log of all the payments made relating to the property. To take a new payment, hit ‘new payment’ in the corner. This will lead you to a page where you can set the fees for the payment and take the card details.

TIP: if you’re running a test property through the system, make sure you are using the correct stripe keys. To learn more about doing test payments, go here

The Manage Tab

The Manage section is where you can update the branch and agent tied to the property and record any sale board or access requirements.

The viewings by field is where you can set which availability is shown on the consumer site. Set this to seller if the seller is handling their own viewings, agent if the agent is handling the viewings, or both if the seller is handling the viewings with the accompaniment of an agent.

At the bottom of the form there’s a ‘withdrawn’ checkbox. Check this if you no longer want the listing to appear on the consumer site.

Listing Section

Presentation

This page shows a summary of the listing.

On the first time viewing this page, there won’t be much to see, but it will be filled out as the listing is put together.

The picture you will be able to see is the featured image- the image shown on the ‘browse properties’ page of the consumer site. To the right of the picture you’ll see the description you have written on the listing details section. Below the image, you’ll see an outline of the rooms included in the listing and the stated price.

At the bottom of the page, there’s a set of booking links. Copy and paste these to clients (i.e. via email) to give them a direct route to book a viewing on the property.

In the top right on the page, you’ll see two buttons: ‘send for approval’ and ‘approve listing for seller’.

Once the listing has been fully composed, hit send for approval to notify the seller. They’ll then be able to look over the listing before it shows on the consumer site. If, for some reason, the seller isn’t able to approve it (if they don’t have access to a computer, for example) hit approve listing for seller to approve it from the staff site.

Once it has been approved, you’ll see a message notifying you underneath the featured image.

Details

The Listing details page is where you’ll edit the main text of the listing. At the top of the page there is a description field. Enter the main property description here- don’t worry, this field will expand as you type (there’s no character limit).

Underneath the description, you’ll see a section for recording the sale status, Price qualifier and asking price. To the right of this you’ll see a set of property fields. It may seem unnecessary to provide this information again, but this is to include the information on the listing. The first time you entered property details was to record the information on the database.

Underneath the property details section, you’ll see a field to add a key feature. Clicking ‘add key feature’ builds the list.

TIP: hit save regularly on this page to make sure nothing is lost if your laptop runs out of power or you lose internet connectivity.

Rooms

A list of all rooms added here will be visible on the listing page on the consumer site. To add a room, click the ‘add room’ button at the top of the page.

On the next page, you’ll see four fields. Anything put into the first two fields will be shown on the consumer site. The ‘room type’ drop down will determine which logo shows next to the room name and dimensions on the consumer listing.

Any text entered the ‘description’ field is included on the published listing on portal sites.
