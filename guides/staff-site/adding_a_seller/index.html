Adding a Seller

You can add sellers to a property by going to the Overview page of the Property & Sellers section. Multiple sellers can be added by clicking the ‘Add seller’ button on the top of the page. Doing so will bring up an empty seller details fields, where you can add the name, email address and contact number for the seller.

Record the seller's details here

To make sure that sellers aren’t accidently duplicated, the system checks to make sure the emails are valid and have not already been recorded elsewhere. If clicking save doesn’t successfully add the seller, please check that the email has correctly been entered.
