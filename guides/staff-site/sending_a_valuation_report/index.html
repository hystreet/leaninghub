Sending a Valuation Report

Once an agent has completed a valuation on the property, the highest, lowest, and recommended values can be recorded on the Valuation Report page. Once the fields have been filled and saved, click the button 'Send report'. This will send the values to the vendor, so they can then make a decision on instructing the agent to sell their home.
