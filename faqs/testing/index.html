Testing the Platform

Can I run test properties?

Yes - you will just need to make sure you are using your Stripe test keys. If you are an Admin user, you can update your Stripe keys by clicking on your initials in the top right corner and going to 'Branches' The boxes to change your Stripe keys are at the bottom of this page. Look for the words ‘test’ or ‘live’ in your keys to tell which keys you are currently using.

If you enter your test keys (usually of the format sk_test_xxxx and pk_test_xxxx) you’ll be able to see how the system handles different responses for you using the test card numbers. For just running through some test customers and doing training the card number you are looking for is 4242 4242 4242 4242. With that card you can use any 3 digit CVC and any expiry date that is in the future.

Using the test keys means that a transaction won't actually be completed but the it will look like a payment has been processed on your test property. This allows you to see the actual process you or a customer would follow to make a payment.

Once you have finished testing, you will need to switch your Stripe keys back to the live codes in order to continue handling real life products. Read our guide on setting up your stripe account to find out more about your Stripe keys.
