Admin Options and Customisation

Do all staff users have access to admin settings?

No. Only users that have been granted admin rights have access to agency and branch settings. Non-admin users will only see the agents and staff pages, and will only be able to modify their own details.

Where cam I find the option to add branches?

Currently, branches can't be added through the staff site platform. If you would like to add new branches, simply get in touch with our support team and we will sort it for you!
