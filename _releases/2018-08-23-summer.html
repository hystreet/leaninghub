Your Summer update from Hystreet

If you've been using Hystreet regularly this month you've probably noticed quite a few updates and changes we've rolled out to make the system a little easier to use day-to-day.

A lot of this is in preparation for some even bigger changes we're planning on rolling out in the final quarter of this year, so watch this space!

See below for more details on everything we've updated:


Overall visual improvements

We have essentially overhauled the way the agent-side of your Hystreet platform looks and feels. Mostly everything is still where you expect it to be, only now it should be a little bit nicer to use.

Clearer admin menu

The 'admin' area is now accessible to users with access to it by clicking the 'admin' button in the top right corner of every page. This makes it a little more accessible, and breaks it off from the 'day-to-day' menu options, which still live in the main menu.

Stay updated at a glance

Property statuses on the property page are now labelled and colour-coded based on status, making it easier for you to see the information you and your staff need straight away.

That's not all...

Everything you've just read are only the updates we're most excited about! Take a look at the list below for a full list of everything else we've updated as well:

  • All lists have had visual improvements and now display better on all mobile devices.
  • Agents can now see and update buyers' and sellers' contact preferences from the agent end of the platform, helping you contact and market to them whilst staying within GDPR regulations.
  • The 'instruction successful' emails have been redesigned as two separate emails, one for for confirmation of the instruction, and one for receipt of payment.
  • Listings can now be published to Zoopla directly from Hystreet.
  • Visual improvements to mobile versions of listings on the client-facing end of the platform.
  • Updated the Stripe payment system for added stability and security on agent and client sites.
  • Text changes on the client site for clarity when agency names are used.
  • Fixed a bug where post-viewing feedback reminders weren't being sent.
  • Fixed a lot of non-user-friendly error messages.

If you want any more information on anything mentioned here, please feel free to get in touch!
